# HytaleLookup

A clean, simple Hytale player lookup page built on the public PlayerDB API. Search by username or UUID, then copy any field with one click.

## Features
- Lookup by username or UUID
- Shareable links using `?q=`
- One-click copy for any field
- Lightweight HTML/CSS/JS
- Responsive layout with subtle animations

## API
Powered by [playerdb.co](https://playerdb.co/). This project only queries the Hytale endpoint.
